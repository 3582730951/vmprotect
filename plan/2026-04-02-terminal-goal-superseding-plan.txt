Threat Model:
- Android `.so`
  - 静态：`strings/readelf/objdump/Ghidra/IDA/JNI registration`
  - 动态：Frida/Zygisk/ptrace/linker hook/inline hook
  - 防护：JNI surface 收敛、导入导出最小化、字符串抽取、anti-hook
- Android `.dex`
  - 静态：JADX/apktool/baksmali/string pool/class name
  - 动态：Frida/Xposed/JDWP/ClassLoader dump/dex dump
  - 防护：encrypted container、loader、method bridge、no plaintext dex
- Android `.ko`
  - 静态：`modinfo/readelf/strings/exported symbols/vermagic`
  - 动态：kprobe/eBPF/module dump/kallsyms
  - 防护：kernel-safe AOT、sign-after-mutate、GKI/KMI gate
- Linux `ELF/.so`
  - 静态：ELF header/symbol table/strings/Ghidra/IDA
  - 动态：`LD_PRELOAD`/ptrace/gdb/uprobe/hook
  - 防护：symbol/import 最小化、section hardening、字符串抽取
- Linux `.ko`
  - 静态：module metadata/exported symbols/strings
  - 动态：kprobe/ftrace/eBPF/module dump
  - 防护：AOT-only、sign-after-mutate、vermagic gate
- iOS
  - 静态：Mach-O sections/Hopper/Ghidra/class-dump/strings
  - 动态：LLDB/Frida/越狱 hook
  - 防护：AOT/解释执行、Mach-O hardening、codesign-aware mutate
- Windows user-mode
  - 静态：PE imports/exports/resources/strings/IDA/Ghidra
  - 动态：x64dbg/WinDbg/ETW/IAT hook/user hook
  - 防护：PE mutate、import 延迟解析、anti-debug/anti-hook、字符串抽取
- Windows `.sys`
  - 静态：dispatch table/IOCTL contract/strings/IDA/Ghidra
  - 动态：KD/WinDbg/Driver Verifier
  - 防护：kernel-safe AOT、dispatch/IOCTL surface 收敛、sign-after-mutate
- `.sh`
  - 静态：文本 grep/教程定位/关键锚点搜索
  - 动态：`set -x`/`source`/子 shell/环境变量泄漏/临时文件恢复
  - 防护：encrypted bundle、launcher、匿名 fd/pipe/memfd、擦除

Hard Constraints:
- iOS App Store
  - 禁止 JIT
  - 禁止 `MAP_JIT`
  - 禁止 RWX
  - 禁止私有 API

- `.sys/.ko`

Performance Strategy:
- 允许：
  - selective virtualization
  - AOT encrypted block
  - direct-threaded interpreter
  - superinstructions
  - resolver cache
  - arena/page pool
- 禁止：
  - 降低 critical 保护强度换性能
  - driver/iOS 使用 JIT/RWX
  - 长期缓存明文字符串、明文 dex、明文脚本
  - 以测试特判绕过安全门
- 预算：
  - user-mode 发布线 `<=10%`
  - iOS 发布线 `<=10%`
  - `.sys/.ko` hot path `<=8%`，整体 `<=12%`
  - `.dex/.sh` 未达预算不得宣称终局完成
